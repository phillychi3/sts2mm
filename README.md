# Slay The Spire 2 Mod Manager

Slay the Spire 2 模組管理工具，支援模組安裝/啟用/停用，以及存檔備份與還原。

## 功能

- **模組管理**：匯入、啟用、停用、卸載模組
- **存檔管理**：備份與還原存檔
- **自動偵測**：自動尋找 Steam 遊戲目錄


## 安裝

### 下載預編譯版本

至 [Releases](https://github.com/phillychi3/sts2mm/releases) 頁面下載對應平台的執行檔，解壓縮後直接執行。



## 使用方式

### 首次設定

1. 將 `sts2mm` 執行檔放在任意目錄後啟動
2. 切換至「設定」頁面，按 `D` 自動偵測遊戲目錄
3. 若偵測失敗，請確認 Steam 已安裝且遊戲已下載完成

### 匯入模組

按 `I` 開啟匯入畫面，將 `.zip` 檔案或模組資料夾**拖入視窗**，或者手動輸入，按 `Enter` 確認安裝。

> 首次安裝模組時，若 mod 存檔槽尚無存檔，程式會詢問是否將一般存檔複製到 mod 槽，以便延續原本進度。
